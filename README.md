# Sales Forecasting (Time Series)

## Project Overview
This project predicts future sales using historical sales data and time series analysis.

## Dataset
Monthly Sales Dataset

## Tools & Technologies
- Python
- Pandas
- Matplotlib
- Statsmodels

## Model Used
- ARIMA

## Output
- 12-month sales forecast
- Actual vs Forecast visualization

## Conclusion
Sales forecasting helps businesses plan inventory, marketing, and demand efficiently.
